ОПИСАНИЕ Леса представляют собой каркасную пространственную систему, изготовленную из трубчатых элементов стоек, балок, стяжек, диагоналей, опор и др. Соединение элементов лесов осуществляется с помощью фиксаторов жёстко закреплённых на балках, стяжках и диагоналях.

Назначение составных частей лесов

Стойки поз. 1 являются основными элементами конструкции и предназначены для восприятия вертикальной нагрузки, а также являются лестницами для подъема и спуска рабочих.

Верхние стойки поз. 2 служат ограждением рабочей площадки.

Раскосы поз. 3 предназначены для обеспечения устойчивости лесов.

Опоры поз. 4 служат для установки на них стоек поз. 1 и регулировки их по высоте.

Рамы поз.5 с помощью своих фиксаторов устанавливаются на необходимой высоте и являются основным элементом, обеспечивающим жесткость конструкции в горизонтальной плоскости. Предназначены для укладки на них настила.

Настил поз.6 является рабочей площадкой.

Стяжки поз.7 являются перилами, ограждающими рабочую площадку.

Диагонали поз.8 служат для обеспечения жесткости при действии боковых (ветровых) нагрузок.

Фиксатор поз.10 представляет собой скобу и предназначен для соединения стоек поз.1 и 2.
